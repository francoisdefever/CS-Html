Structuration 

<!DOCTYPE html> = grammaire générale des balises html
                        
 <html> = début de la page
                        
 <head> = Entête qui ne s'affuiche pas sur le site
                        
 <meta charset="utf-8" /> = pour afficher les accents
                        
 <title>Page de base</title>= titre de la page, il s'affiche dans l'onglet du navigateur
                        
 </head>
                        
 <body> = Corps qui s'affiche sur la page
                        
 </body>
                        
 </html> = Fin de page

Balises

Balises Sémantiques
  • <article>Cette balise définit un contenu externe.(exemple billet de blog)
  • <aside>Cette balise définit un contenu mis à part (comme une zone de pub par exemple).
  • <blockquote>Citation (longue)
  • <code>Code technique informatique
  • <dd>Définition du terme
  • <h1>Titre de niveau 1
  • <head>En-tête de la page
  • <img>Pour insérer une image
  • <video>Pour insérer une vidéo
  • <hr />Ligne de séparation horizontale
Balises Sémantiques
  • <p>Paragraphe
  • <pre>Affichage formaté (pour les codes sources)
  • <script>Insertion code JavaScript
  • <table>Pour insérer un Tableau
  • <tbody>Section du corps du tableau
  • <td>Pour insérer une Cellule
  • <time>Date ou heure
  • <ul>Liste à puces, non numérotée
  • <ol>Liste numérotée
  • <li>Élément de la liste à puces
  • <nav>Définit une section de navigation
